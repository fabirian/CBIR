Para ejecutar, abrir el cmd en la carpeta del proyecto
Primero se ejecuta

pip install -r requirements.txt

Para instalar librerias necesarias
Luego se ejecuta 

python init.py 

para cargar en los archivo que se encuentran en /config
las caracteristicas del dataset

Despues puede ejecutar

python runner.py

Para lanzar la pagina web en el localhost, cuando runner.py se este ejecutando
abre el navegador y usa la url http://localhost:5000/ para visualizar la interfaz de la aplicacion
